# Retail Sales Dashboard

## Objective

Analyze retail sales performance through an interactive dashboard.

## Dataset

Sample Superstore Dataset

## Tools

- LibreOffice Calc

## Dashboard Includes

- Total Sales
- Total Profit
- Sales by Category
- Profit by Region
- Quantity Analysis
- Segment Performance

## Business Insights

- West region generated the highest sales.
- Consumer segment generated the highest profit.
- Technology category contributed the highest revenue.
