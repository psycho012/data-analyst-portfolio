# SQL Sales Analysis

## Objective

Analyze retail business performance using SQL.

## Database

SQLite

## Skills Demonstrated

- Data Retrieval
- Data Filtering
- Aggregate Functions
- Business Analysis
- Reporting

## SQL Concepts

- SELECT
- WHERE
- LIMIT
- DISTINCT
- AS
- GROUP BY
- HAVING
- ORDER BY

## Business Insights

- West region achieved the highest sales.
- Technology generated the highest profit.
- Consumer segment had the highest average sales.
- Multiple states recorded negative profit.
